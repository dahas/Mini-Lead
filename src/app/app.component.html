Virtual Analog Synthesizer - build with Web Technologies

The above synthesizer is an example to demonstrate a small part of what is possible with JavaScript/TypeScript, HTML and CSS nowadays. It is build with the Angular framework and uses basic features of the Web Audio Api. It is not intended to work on all web browsers, but at least on Google Chrome. Use your keyboard or mouse pointer to play with it.

The source code of the project can be found on GitHub.
